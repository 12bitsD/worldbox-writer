# WorldBox Writer 质量评估框架

## 北极星
最终目标: 让 Agent 写出 [对标定义] 级别的小说。

每轮开始前回答 3 个问题:
1. 本轮选择的差距,是否直接影响"小说写作质量"?
2. 本轮的可验证标准是什么?(必须是数字或 pass/fail)
3. 如果本轮成功,小说质量在哪个维度提升?

任一回答模糊 → 重新选题。

---

## 双轴定义(贯穿四档)

### 轴 A: 故事力(是否精彩)

评估要点(每点 1–10 分,盲评):
- **钩子**: 前 500 字是否让评委想继续读
- **冲突密度**: 每千字核心冲突/张力点数量
- **反转有效性**: 伏笔→误导→揭示链路成立率
- **人物动机一致性**: 人物决策是否可被读者倒推解释
- **悬念维持**: 读到 30%/60%/90% 处的"想继续读"率
- **结构完整性**: 开端-发展-高潮-结局的节奏是否塌掉

### 轴 B: 文笔

评估要点(每点 1–10 分,盲评 + 客观指标双轨):
- **句式多样性** (客观: 句长方差、句型分布)
- **意象/比喻新鲜度** (主观: 陈词滥调命中率,越低越好)
- **描写精准度** (主观: 场景/动作/情绪是否"立得住")
- **节奏感** (主观: 快慢交替是否得当)
- **对话辨识度** (主观: 遮住人名能否猜出谁在说话)
- **信息密度** (主观: 有无注水/废句)

---

## 评测协议(四档共用)

1. **样本**: 每档跑 N=20 篇定 prompt 生成的文本,长度按档位要求
2. **盲评**: 至少 3 名评委独立打分,去掉最高最低取均值
3. **对标盲测**: 同 prompt 由对标对象生成,A/B 随机顺序,评委选择更"精彩/文笔更好"的一篇,得"胜率"
4. **客观指标**: 每篇跑文本分析脚本,产出句长方差、TTR(词汇丰富度)、陈词滥调命中率
5. 任一档进入下一档前,所有指标必须达标且至少跑 2 轮稳定不回退

---

## 四档标准(双轴并行)

### 🟫 L1 追平市面(对标: 起点中位网文 / Sudowrite 默认输出)
- 样本: 5 篇短篇(每篇 5k–8k 字)
- 故事力: 盲评均分 ≥ 6.0/10; "读完不弃"率 ≥ 70%
- 文笔: 盲评均分 ≥ 6.0/10; 陈词滥调命中率 ≤ 8%
- 盲测胜率(vs 起点中位作品): ≥ 40%
- **退出门槛**: 无明显逻辑断裂、无人称错乱、无"AI 味"明显段落(评委一致识别为 AI 的比例 ≤ 50%)

### 🟦 L2 好(对标: 起点头部签约作 / 一般传统出版小说)
- 样本: 3 篇中篇(每篇 3w–5w 字)
- 故事力: 盲评均分 ≥ 7.0/10; 读者读到 60% 处"想继续"率 ≥ 70%
- 文笔: 盲评均分 ≥ 7.0/10; 对话辨识度盲猜准确率 ≥ 60%
- 盲测胜率(vs 起点头部): ≥ 45%
- **退出门槛**: 能稳定写出至少 2 个反转(伏笔成立); 主线无塌方

### 🟩 L3 优秀(对标: 头部网络作家 如猫腻/烽火 / 当代严肃文学新人作家)
- 样本: 1 部长篇(15w–25w 字,可分卷生成但需整体一致)
- 故事力: 盲评均分 ≥ 8.0/10; 主角动机一致性评委一致认可 ≥ 80%; 长篇内反转成立 ≥ 5 处
- 文笔: 盲评均分 ≥ 8.0/10; 陈词滥调命中率 ≤ 3%; 意象新鲜度评委正向标注 ≥ 30 处/10w 字
- 盲测胜率(vs 头部网络作家或新人纯文学): ≥ 50%
- **退出门槛**: 世界观自洽(无前后矛盾被评委指出 ≥ 3 处); 至少 1 个配角"立得住"

### 🟥 L4 无敌(对标: 顶级作家 余华/麦家级 或 Claude/GPT 当前最佳输出)
- 样本: 1 部长篇(40w+ 字) + 5 篇风格各异的短篇
- 故事力: 盲评均分 ≥ 9.0/10; 多线叙事 ≥ 3 条且收束成立
- 文笔: 盲评均分 ≥ 9.0/10; 遮去作者后,评委误判为人类作家比例 ≥ 70%
- 盲测胜率: vs Claude/GPT 当前最好输出 ≥ 50%; vs 一线人类作家选段 ≥ 35%
- **退出门槛**: 任意评委自发使用"动人/难忘/想再读一遍"等强情感词的比例 ≥ 50%

## 快速评测方法(每轮迭代用)

Agent 自动迭代时无法每轮请真人评委。采用:
- **每轮**: LLM-as-judge (用强模型当评委 + 本 rubric) 做快速评测
- **跨档前**: 上真人评委做一次校准,防止 LLM 评委自我循环偏差

### LLM-as-judge Prompt 模板

```
你是一位资深中文小说编辑。请对以下小说片段进行评估。

评分维度(每项 1-10):
1. 钩子: 前 500 字是否让你想继续读?
2. 冲突密度: 每千字有几个核心冲突/张力点?
3. 人物动机一致性: 人物决策是否可被倒推解释?
4. 句式多样性: 句子长短和结构是否有变化?
5. 意象新鲜度: 有没有让你眼前一亮的比喻或描写?
6. 对话辨识度: 遮住人名,能猜出谁在说话吗?
7. 信息密度: 有没有注水或废句?
8. AI味: 读起来像人写的还是AI写的?(1=明显AI, 10=完全像人)

输出 JSON:
{"scores": {...}, "overall": X, "best_line": "...", "worst_line": "...", "one_line_suggestion": "..."}
```

---

## 迭代规则

每轮迭代必须能在双轴(故事力 + 文笔)中至少一轴产生可量化的提升,否则不算有效迭代,计入"连续无提升"轮次,触发停机条件(3 轮连续无提升 → 停止迭代,重新评估方向)。
